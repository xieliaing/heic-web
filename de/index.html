    Häufig gestellte Fragen

    Was ist eine HEIC-Datei?

    HEIC (High Efficiency Image Container) ist das Standard-Fotoformat auf iPhones und iPads seit iOS 11. Es bietet bei ähnlicher Qualität eine bessere Komprimierung als JPEG, wird jedoch auf Windows, Websites oder älterer Software nicht weit verbreitet unterstützt — weshalb die Konvertierung in JPG oder PNG oft notwendig ist.

    Werden meine Fotos irgendwo hochgeladen?

    Nein. Dieser Konverter läuft vollständig in Ihrem Webbrowser mit einer WebAssembly-Version von libheif. Ihre Dateien werden niemals an einen Server gesendet. Sie können die Internetverbindung nach dem Laden der Seite sogar trennen, und es wird trotzdem funktionieren.

    Gibt es eine Dateigrößen- oder Anzahlbeschränkung?

    Es gibt keine harte Begrenzung, aber da die Konvertierung den Arbeitsspeicher Ihres Computers verwendet, können sehr große Dateien (100+ MB) oder sehr große Stapel Ihren Browser verlangsamen. Für beste Ergebnisse konvertieren Sie Dateien in Stapeln von 10–20 auf einmal.

    JPG, PNG oder WebP — welches soll ich wählen?

    Wählen Sie JPEG für Fotos, die Sie teilen oder per E-Mail senden möchten — es wird überall unterstützt und erzeugt kleine Dateien. Wählen Sie PNG, wenn Sie verlustfreie Qualität oder Transparenz benötigen. Wählen Sie WebP, wenn Sie auf eine Website hochladen und die kleinste Dateigröße bei gleicher Qualität wünschen (typischerweise 25–35% kleiner als JPEG). Für die meisten iPhone-Fotos ist JPEG die sichere Standardwahl.
